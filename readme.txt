A simple widget which displays a configurable list of the latest Betting News.

== Description ==

This plugin fetches all relevant publicly available news feeds of Betting News. The news can be placed on the blog through a highly configurable widget.
The widget allows you to edit the entire formatting of each RSS item, such as description, count, title and even how the list should appear and the link tags themselves.

== Installation ==

1. Upload the folder `betting-news` to the `/wp-content/plugins/` directory or search for `Betting News` in `Plugins` > `Add new` and click `Install` to the right of `Betting News`
2. Activate the plugin through the `Plugins` menu in WordPress
